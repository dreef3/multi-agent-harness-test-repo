Implementation Plan: Task-by-task plan to analyze E2E Test Repo (f648b34b-f510-4880-9f8e-9f254b32ab5d)

Overview
- The implementation will be a sequence of independent, self-contained tasks. Each task includes exact shell commands to run in the repository root and the expected output files to create under reports/. Tasks are non-invasive (read-only). Estimated time per task given.

Preparation (common instructions)
- Run from the repository root (default branch main checked out by harness).
- Ensure standard CLI tools are available: bash, find, ls, sed, awk, jq, tree (if available), rg or grep, python (for JSON helpers). If some tools are missing, use portable equivalents (find + sed + awk).
- Create reports/ directory to hold artifacts: mkdir -p reports

Task 1 — File index
- Purpose: Create a recursive file listing with metadata (path, size, mtime, type)
- Commands:
  mkdir -p reports
  find . -path './.git' -prune -o -type f -print0 | xargs -0 stat --format='%n|%s|%Y' > reports/file-list.txt || \
  (find . -path './.git' -prune -o -type f -print | while read f; do echo "$(wc -c <"$f")|$(stat -c %Y "$f")|$f"; done) > reports/file-list.txt
  # Convert to JSON
  python - <<'PY'
import json,sys
out=[]
for line in open('reports/file-list.txt'):
    try:
        parts=line.strip().split('|')
        if len(parts)==3:
            size,mtime,path=parts[0],parts[1],parts[2]
        else:
            size,mtime,path = parts[0],None,parts[-1]
        out.append({'path':path,'size':int(size),'mtime':mtime})
    except Exception:
        pass
json.dump(out,open('reports/file-index.json','w'),indent=2)
PY
- Output: reports/file-index.json
- Est. time: 1-2 minutes

Task 3 — Detect languages & manifests
- Purpose: Detect languages and tooling by searching for common manifest files and file extensions
- Commands:
  # Look for common manifests
  rg --hidden --no-ignore-vcs -n "^(name|version)" package.json || true
  for f in package.json pyproject.toml setup.py requirements.txt Pipfile requirements*.txt go.mod Cargo.toml pom.xml build.gradle composer.json composer.lock; do [ -f "$f" ] && echo "$f"; done > reports/manifests-found.txt
  # Also detect CI
  rg --hidden --no-ignore-vcs "workflow" .github -S || true
  # Language heuristics
  find . -path './.git' -prune -o -name '*.py' -print | head -n 5 > reports/_sample_python.txt || true
  find . -path './.git' -prune -o -name '*.js' -o -name '*.ts' -print | head -n 5 > reports/_sample_js.txt || true
  find . -path './.git' -prune -o -name '*.go' -print | head -n 5 > reports/_sample_go.txt || true
- Post-process to JSON
  python - <<'PY'
import json
man=[]
try:
    man=open('reports/manifests-found.txt').read().strip().splitlines()
except FileNotFoundError:
    man=[]
samples={}
for name in ('_sample_python.txt','_sample_js.txt','_sample_go.txt'):
    try:
        samples[name]=open('reports/'+name).read().strip().splitlines()
    except FileNotFoundError:
        samples[name]=[]
json.dump({'manifests':man,'samples':samples},open('reports/language-summary.json','w'),indent=2)
PY
- Output: reports/language-summary.json, reports/manifests-found.txt
- Est. time: 2-5 minutes

- Output: reports/manifest-*.txt
- Est. time: 1-3 minutes

Task 5 — Dependency snapshot
- Purpose: Produce a simple dependency snapshot (best-effort) by reading manifests and lockfiles
- Commands (examples):
  if [ -f package.json ]; then jq '.dependencies,.devDependencies' package.json > reports/npm-deps.json || true; fi
  if [ -f requirements.txt ]; then awk 'NF && $1!="#"{print $1}' requirements.txt > reports/requirements-parsed.txt || true; fi
  if [ -f go.mod ]; then cat go.mod > reports/go-mod.txt; fi
  if [ -f Cargo.toml ]; then cat Cargo.toml > reports/Cargo-toml.txt; fi
- Output: reports/npm-deps.json, reports/requirements-parsed.txt, reports/go-mod.txt, reports/Cargo-toml.txt
- Est. time: 2-5 minutes

Task 9 — Generate human-readable repository-structure.md
- Purpose: Combine all findings into a concise Markdown summary
- Commands (generate with Python script):
  python - <<'PY'
import json,os
out=[]
ri='reports'
# Load file index
try:
    fi=json.load(open(os.path.join(ri,'file-index.json')))
except Exception:
    fi=[]
# Load language summary
lsum={}
try:
    lsum=json.load(open(os.path.join(ri,'language-summary.json')))
except Exception:
    pass
man=[]
if os.path.exists(os.path.join(ri,'manifests-found.txt')):
    man=open(os.path.join(ri,'manifests-found.txt')).read().strip().splitlines()
# Compose markdown
md=[]
md.append('# Repository structure summary')
md.append('\n## Top-level summary')
md.append(f'- Total files scanned: {len(fi)}')
if 'manifests' in lsum and lsum['manifests']:
    md.append('\n## Detected manifests')
    for m in lsum['manifests']:
        md.append(f'- {m}')
if man:
    md.append('\n## Manifests found (raw)')
    for m in man:
        md.append(f'- {m}')
md.append('\n## Samples (first few files by language)')
for k,v in lsum.get('samples',{}).items():
    md.append(f'### {k}\n')
    for s in v:
        md.append(f'- {s}')
md.append('\n## Recommendations')
md.append('- Inspect reports/ for full artifacts.\n- If you want automated dependency graphs or license scanning, run specialized tools (e.g., depcheck, pipdeptree, go list -m all, cargo metadata).')
open('reports/repository-structure.md','w').write('\n'.join(md))
print('wrote reports/repository-structure.md')
PY
- Output: reports/repository-structure.md
- Est. time: 1-3 minutes
