Task 3 — Language detection & LOC (2-5 min)
- Purpose: detect programming languages and approximate LOC per language.
- Preferred tools: tokei or cloc. Fallback: simple file extension counts.
- Commands (try preferred then fallback):
  cd workspace/repo
  if command -v tokei >/dev/null 2>&1; then tokei --output json . > ../../reports/.../languages.json; elif command -v cloc >/dev/null 2>&1; then cloc --json . > ../../reports/.../cloc.json; else (find . -type f | sed 's|.*\.||' | sort | uniq -c | sort -nr) > ../../reports/.../extensions_count.txt; fi
- Outputs:
  reports/.../languages.json or cloc.json or extensions_count.txt

Task 4 — Detect package managers and dependency manifests (3-5 min)

- Commands:
  cd workspace/repo
  rg --hidden --glob '!**/.git/*' -n "(^|/)package.json$|(^|/)pyproject.toml$|(^|/)requirements.txt$|(^|/)go.mod$|(^|/)Cargo.toml$|(^|/)pom.xml$|(^|/)Gemfile$|(^|/)Pipfile$|(^|/)composer.json$|(^|/)package-lock.json$|(^|/)yarn.lock$|(^|/)pnpm-lock.yaml" --files || find . -type f \( -name "package.json" -o -name "pyproject.toml" -o -name "requirements.txt" -o -name "go.mod" -o -name "Cargo.toml" -o -name "pom.xml" -o -name "Gemfile" -o -name "Pipfile" -o -name "composer.json" -o -name "package-lock.json" -o -name "yarn.lock" -o -name "pnpm-lock.yaml" \) > ../../reports/.../dependency_manifests.txt

Task 10 — Produce human-friendly summary and structured JSON (5-10 min)
- Purpose: synthesize findings into structure.md and structure.json
- Procedure (example script):
  cd workspace
  python3 - <<'PY'
import json,os
rid='45bc0ff1-05dc-4210-904a-f2c288471d83'
base='reports/'+rid
out_md=base+'/structure.md'
out_json=base+'/structure.json'

summary={}
# populate summary from files generated above if they exist
files_of_interest=['top_level.txt','tree_L2.txt','languages.json','cloc.json','dependency_manifests.txt','ci_candidates.txt','test_dirs.txt','dockerfiles.txt','largest_files.txt','meta_docs.txt']
for f in files_of_interest:
    p=os.path.join(base,f)
    if os.path.exists(p):
        with open(p,'r',encoding='utf-8',errors='ignore') as fh:
            summary[f]=fh.read()[:10000]

res={
  'repo_id':rid,
  'overview':'Auto-generated summary — see fields for excerpts',
  'artifacts':{k:os.path.exists(os.path.join(base,k)) for k in files_of_interest}
}
with open(out_json,'w') as jh: json.dump(res,jh,indent=2)
with open(out_md,'w') as mh:
    mh.write('# Repository structure summary\n')
    mh.write('\nRepo id: '+rid+'\n\n')
    mh.write('Artifacts present:\n')
    for k,v in res['artifacts'].items(): mh.write('- %s: %s\n'%(k,str(v)))
    mh.write('\nSee individual files in this folder for details.\n')
PY
- Outputs:
  reports/.../structure.md
  reports/.../structure.json
