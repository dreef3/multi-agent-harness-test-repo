Implementation Plan: Codebase Structure Analysis for E2E Test Repo (08088a6f-4199-4f4d-8084-1e6b2eb6ccf9)

Overview
- This plan contains a set of self-contained tasks a sub-agent can run after cloning the repository. Each task includes: purpose, exact commands, expected output files (paths inside the workspace), and checks to ensure success.
- Repository root is assumed at /workspace/repo (sub-agent should clone into that path before running tasks).

Common preconditions (run once)
1. Clone repository into /workspace/repo
   - git clone https://github.com/dreef3/multi-agent-harness-test-repo.git /workspace/repo
   - cd /workspace/repo
2. Install common CLI tools (if not present): tree, jq, rg (ripgrep), cloc
   - apt-get update && apt-get install -y tree jq ripgrep cloc || use package-manager appropriate to agent environment

Task 1 — Top-level inventory
- Purpose: capture top-level files and directories
- Commands:
  - cd /workspace/repo
  - find . -maxdepth 2 -mindepth 1 -print | sed 's|^./||' > /workspace/artifacts/file_tree.txt
  - tree -L 2 -a -I 'node_modules|venv' > /workspace/artifacts/tree_L2.txt || true
- Expected outputs:
  - /workspace/artifacts/file_tree.txt
  - /workspace/artifacts/tree_L2.txt
- Success checks: files created and non-empty

Task 2 — Language detection and LOC
- Purpose: determine languages and lines of code
- Commands:
  - cd /workspace/repo
  - cloc --json --out=/workspace/artifacts/cloc.json . || (echo '{}' > /workspace/artifacts/cloc.json)
  - git ls-files | sed -n '1,200p' > /workspace/artifacts/git_files_sample.txt
- Expected outputs:
  - /workspace/artifacts/cloc.json (JSON with language LOC)
  - /workspace/artifacts/git_files_sample.txt

Task 3 — Identify package manifests and dependency lists
- Purpose: find dependency manifest files and extract declared dependencies
- Commands (run each if file exists):
  - cd /workspace/repo
  - for f in package.json requirements.txt requirements-dev.txt Pipfile pyproject.toml setup.py setup.cfg Cargo.toml go.mod build.gradle pom.xml; do if [ -f "$f" ]; then echo "FOUND:$f" >> /workspace/artifacts/manifests.txt; fi; done
  - If package.json exists: jq -S '.dependencies, .devDependencies' package.json > /workspace/artifacts/package_json_deps.json || true
  - If pyproject.toml exists: cat pyproject.toml > /workspace/artifacts/pyproject.toml
  - If requirements.txt exists: cp requirements.txt /workspace/artifacts/requirements.txt
  - If Cargo.toml/go.mod/pom.xml/build.gradle exist: copy them to /workspace/artifacts/
- Expected outputs:
  - /workspace/artifacts/manifests.txt
  - /workspace/artifacts/package_json_deps.json or copied manifest files
- Notes: For deeper dependency resolution, run language-specific dep tools (npm ci && npm ls --json, pip install -r requirements.txt && pipdeptree --json). These are optional and may require network access.

Task 4 — CI and automation inspection
- Purpose: summarize CI configs and workflow triggers
- Commands:
  - cd /workspace/repo
  - find .github -type f -maxdepth 3 -print > /workspace/artifacts/ci_files.txt || true
  - for wf in .github/workflows/*.yml .github/workflows/*.yaml; do if [ -f "$wf" ]; then echo "--- $wf ---" >> /workspace/artifacts/ci_summary.md; sed -n '1,200p' "$wf" >> /workspace/artifacts/ci_summary.md; fi; done || true
  - Also copy Dockerfile(s): find . -iname Dockerfile -print > /workspace/artifacts/dockerfiles.txt
- Expected outputs:
  - /workspace/artifacts/ci_files.txt
  - /workspace/artifacts/ci_summary.md
  - /workspace/artifacts/dockerfiles.txt

Task 5 — Tests and test commands
- Purpose: locate test frameworks and capture how tests run
- Commands:
  - cd /workspace/repo
  - rg "pytest|unittest|mocha|jest|ava|go test|mvn test|gradle test|cargo test" -n --hidden --no-ignore-vcs || true > /workspace/artifacts/test_patterns.txt
  - Check package.json for test script: jq -r '.scripts.test // empty' package.json > /workspace/artifacts/npm_test_script.txt || true
  - If Makefile exists: rg "test:" Makefile -n || true > /workspace/artifacts/makefile_test_targets.txt
- Expected outputs:
  - /workspace/artifacts/test_patterns.txt
  - /workspace/artifacts/npm_test_script.txt
  - /workspace/artifacts/makefile_test_targets.txt

Task 6 — Documentation, license, contribution
- Purpose: capture README, CONTRIBUTING, LICENSE
- Commands:
  - cd /workspace/repo
  - for f in README.md README.rst CONTRIBUTING.md LICENSE; do if [ -f "$f" ]; then cp "$f" /workspace/artifacts/; fi; done
- Expected outputs:
  - /workspace/artifacts/README.md (if present)
  - /workspace/artifacts/CONTRIBUTING.md
  - /workspace/artifacts/LICENSE

Task 7 — Static risk scan (quick)
- Purpose: identify obvious risk items: pinned dependencies, outdated CI secrets, Docker latest tags
- Commands (heuristic searches):
  - cd /workspace/repo
  - rg "\b(password|secret|AWS|GCP|TOKEN|PASSWORD|SECRET|API_KEY)\b" --hidden --no-ignore-vcs -S -n || true > /workspace/artifacts/secrets_hits.txt
  - rg ":\s*latest\b|FROM .*:latest|apt-get upgrade -y" --hidden --no-ignore-vcs -S -n || true > /workspace/artifacts/docker_latest_hits.txt
  - rg "==|===|@latest" --hidden --no-ignore-vcs -S -n || true > /workspace/artifacts/pinned_version_hits.txt
- Expected outputs:
  - /workspace/artifacts/secrets_hits.txt
  - /workspace/artifacts/docker_latest_hits.txt
  - /workspace/artifacts/pinned_version_hits.txt
- Notes: These are heuristic; a human should triage findings.

Task 8 — Summarize and produce reports
- Purpose: produce human-readable summaries and a short risk report
- Commands:
  - cd /workspace
  - python3 - <<'PY'
import json,sys
art='/workspace/artifacts'
# Minimal summarizer reading cloc and manifests
try:
    cloc=json.load(open(art+'/cloc.json'))
except Exception:
    cloc={}
manifests = open(art+'/manifests.txt').read() if os.path.exists(art+'/manifests.txt') else ''
summary = {
  'cloc': cloc,
  'manifests': manifests[:4000],
}
open(art+'/summary.json','w').write(json.dumps(summary,indent=2))
print('WROTE',art+'/summary.json')
PY
- Expected outputs:
  - /workspace/artifacts/summary.json
  - /workspace/artifacts/analysis_report.md (create a small markdown summarizing key findings by assembling earlier artifacts)

- Expected outputs:
  - /workspace/artifacts/deps.svg (optional)
